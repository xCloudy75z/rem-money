Icon Generator.

Spending Tracker · One-click PNG export

Renders the chosen logo (Amiri dirham on cream paper, accent rule) at the four sizes iOS & Android home screens actually need, then downloads them as PNGs. Run this once after any logo edit.

  1. Click the button above. Wait ~2 seconds (waits for the Amiri font to load).
  2. Your browser will download four files in quick succession. If asked to "Allow multiple downloads," click Allow.
  3. Move all four into v2/icons/ (the folder next to icon.svg).
  4. Back in the terminal: npm run build. The icons are now in dist/icons/ and referenced by the manifest.
  5. Re-drag the dist/ folder onto Netlify Drop. Reinstall on phone — icon should show, not white.
